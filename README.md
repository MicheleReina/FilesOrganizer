# FilesOrganizer

Rearrange files in folders according to txt folder filename rules and initial name of the files

ex txt filename rules: 

| folder name |
| ------------- |
| aaa |
| bbb |
| ccc |

ex files :
| files name |
| ------------- | 
| aaa_57y498 | 
| bbb-8945735 |
| ccc hbdgsyihgbv |


The program should rearrange files in that way

| files name | folder to move | folder |
| ------------- | ------------- |------------- |
| aaa_57y498 | --> |aaa |
| 8945735 | --> | bbb |
| ccc hbdgsyihgbv | --> | ccc |
